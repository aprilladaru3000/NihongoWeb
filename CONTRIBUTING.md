# Contributing

Thanks for your interest in contributing to this project! Small, focused contributions are welcome.

Before you start

- Fork the repository and clone your fork.
- Create a new branch with a descriptive name, e.g. `feature/admin-modal` or `fix/delete-bug`.

Make changes

- Run the server locally when working on backend or full-stack changes:

```powershell
node server.js
```

- Edit files under `public/` for frontend changes.

Commit

- Write clear commit messages. Example:

```
git commit -m "Add admin modal with reply button"
```

Push and create PR

- Push your branch to your fork and open a Pull Request against the `main` branch of the upstream repository.

Coding style

- Keep HTML simple and semantic.
- Use Tailwind CSS utilities for styling.
- Keep JavaScript vanilla and minimal for this project.

Contact

If you have a question, open an issue in the repository.
